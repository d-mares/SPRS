About Booky

A user-friendly C# Windows Forms application designed to simulate the functionality of a modern bookstore This project connects to a robust, locally hosted, database to manage and display bookstore operations, offering a practical solution for displaying book inventories and handling user interactions, all while offering recommendations based on what the user interacts with.


Purpose

Booky was created as my final project for an application development course, providing an opportunity to apply and integrate the concepts learned throughout the curriculum. The project was designed to deepen understanding of:

  • Connecting a C# application to a database and dynamically managing data.
  • Implementing event-driven programming to create a responsive and user-friendly application.
  • Designing and organizing an intuitive interface suitable for a real-world scenario.

Beyond fulfilling the course requirements, Booky became a platform for personal growth and skill-building. It allowed for hands-on experience with database management, Windows Forms development, and problem-solving in software design. The project represents both a culmination of academic learning and a stepping stone for future, more ambitious endeavors.


Key Features

User Management
  • Store user purchase history in an SQL database.
  • Simple user authentication for login.
Product Management
  • Add, edit, delete, and view products with details (ID, name, category, price)
  • Use SQL to store and manage product data.
Purchase Management
  • Enable users to simulate purchases, updating purchase history and stock.
  • Save each purchase transaction to the SQL database.
Personalized Recommendations
  • Use simple data analysis (e.g., most frequently purchased categories, recently purchased items) to suggest products.
  • These recommendations are based on basic statistics from the purchase data, without complex AI models.
Database Management
  • Use SQL to manage user data, product data, and purchase transactions
  • Data persistence to ensure all data is saved and can be retrieved.
Reports
  • Generate a summary report showing purchase history for each user
  • Report on most popular products and categories.
